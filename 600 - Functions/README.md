## Python decorators
```python
def deco(f):
    print('Decorator starts')

    @wraps(f)
    def inner_func(*args, **kwargs):
        print(f'Inner starts')
        result = f(*args, **kwargs)
        print(f'Inner ends')
        return result

    print('Decorator ends')
    return inner_func


@deco('paramter value')
def simple_print_args(arg1, arg2):
    print(arg1)
    print(arg2)
```
